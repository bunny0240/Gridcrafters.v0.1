Update the typography system across the entire GridCrafters app.

---

## FONT IMPORTS

Add this to the top of your global CSS file (index.css or globals.css):

@import url('https://fonts.googleapis.com/css2?family=Geist:wght@300;400;500;600;700;800&family=DM+Mono:wght@400;500&display=swap');

---

## CSS VARIABLES

Replace all existing font-family variables with:

:root {
  --font-sans: 'Geist', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
  --font-mono: 'DM Mono', 'JetBrains Mono', 'Fira Code', monospace;
}

---

## GLOBAL BASE

Set on body and all elements:

body {
  font-family: var(--font-sans);
  font-size: 13px;
  line-height: 1.6;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
  font-feature-settings: 'cv02', 'cv03', 'cv04', 'cv11';
}

* {
  font-family: inherit;
}

## RULES — do not violate these

1. NEVER use font-weight 900. Max is 700 for display, 600 for headings,
   500 for emphasis, 400 for body.

2. NEVER use letter-spacing tighter than -0.5px or wider than 1.2px
   (only on uppercase micro-labels).

3. NEVER apply Geist to spreadsheet cells, formula bars, shortcut key
   caps, or any data numbers — those always use DM Mono.

4. NEVER mix font families within the same UI component.
   One component = one font (either Geist or DM Mono, never both).

5. Body text line-height is always 1.6. Heading line-height is always 1.1–1.2.

6. Uppercase labels (section separators, eyebrows) use Geist 600 at
   9.5px with 1px letter-spacing. Do not exceed 10px for these labels.

7. The brand name in the sidebar uses font-weight 700 with
   letter-spacing -0.4px. The word "Crafters" gets the purple→blue
   gradient. "Grid" stays plain white #e8e6e0.

8. Remove all uses of Playfair Display, DM Sans, system-ui stacks,
   Nunito, or any other font previously used. Geist replaces everything
   except cells and data which use DM Mono.

9. font-feature-settings: 'cv02','cv03','cv04','cv11' must be set on
   body — these activate Geist's alternate glyphs that remove the
   childish circular letter shapes.

10. -webkit-font-smoothing: antialiased must be set on body.
    Without it Geist looks slightly blurry on macOS.

## TAILWIND CONFIG (if using Tailwind)

In tailwind.config.js, extend fontFamily:

module.exports = {
  theme: {
    extend: {
      fontFamily: {
        sans: ['Geist', '-apple-system', 'BlinkMacSystemFont', 'sans-serif'],
        mono: ['DM Mono', 'JetBrains Mono', 'monospace'],
      },
    },
  },
}

Then in index.css:

@layer base {
  body {
    @apply font-sans antialiased;
    font-feature-settings: 'cv02', 'cv03', 'cv04', 'cv11';
  }
}
